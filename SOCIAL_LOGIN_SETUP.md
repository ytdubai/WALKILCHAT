## THE STRATEGY

Users login with Twitter/Instagram/LinkedIn → We get permission to post on their behalf → Viral growth!

## SETUP REQUIRED:

### 1. Twitter/X Developer Account

**Go to:** https://developer.twitter.com

**Steps:**
1. Apply for developer account (Free tier or $100/month Basic)
2. Create new App: "WakilChat"
3. Enable OAuth 2.0
4. Set permissions: Read + Write (to post tweets)
5. Add callback URL: `https://wakilchat.com/auth/twitter/callback`
6. Get: Client ID + Client Secret

**Cost:** Free (or $100/month for more posts)

### 2. Instagram/Meta Business

**Go to:** https://developers.facebook.com

**Steps:**
1. Create Meta App
2. Add Instagram Basic Display
3. Add permissions: instagram_basic, instagram_content_publish
4. Callback URL: `https://wakilchat.com/auth/instagram/callback`
5. Get: App ID + App Secret

**Cost:** Free

### 3. LinkedIn API

**Go to:** https://www.linkedin.com/developers

**Steps:**
1. Create LinkedIn App
2. Request access to Marketing Developer Platform
3. Permissions: w_member_social (to post)
4. Callback: `https://wakilchat.com/auth/linkedin/callback`
5. Get: Client ID + Client Secret

**Cost:** Free
