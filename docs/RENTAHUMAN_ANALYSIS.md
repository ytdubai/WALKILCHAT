# RentAHuman.ai Business Model Analysis 🤖

## What They Do

**Concept:** AI agents hire humans to do physical tasks they can't do remotely.

### Their Model:

**Supply Side (Humans):**
- Sign up with skills, location, hourly rate
- Wait for AI agents to book them
- Complete physical tasks
- Get paid in stablecoins (crypto)

**Demand Side (AI Agents):**
- Use MCP (Model Context Protocol) or REST API
- Search for humans by location/skill
- Book and pay automatically
- Get task completed

**Tasks:**
- Package pickups
- In-person meetings
- Document signing
- Physical verification
- Event attendance
- Hardware setup
- Real estate viewing
- Lab testing
- Photos of locations
- Purchases/shopping

**Revenue:**
- Take 15-20% commission on each task
- Subscription for premium AI agent access
- API usage fees
